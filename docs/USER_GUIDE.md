# Руководство пользователя VCAD

## Навигация в 3D

| Действие | Управление |
|----------|------------|
| Вращение камеры | Средняя кнопка мыши (перетаскивание) |
| Вращение камеры (альт.) | Alt + левая кнопка мыши |
| Масштабирование | Колёсико мыши |
| Панорамирование | Shift + средняя кнопка мыши |

---

## Выделение

### Выделение объекта
- **Клик** на объект — выделить
- **Ctrl+клик** — добавить/убрать из выделения
- **Клик** в пустое место — снять выделение

### Выделение грани
- **Shift+клик** на объект — выделить грань
- Грань подсвечивается оранжевым цветом
- В панели свойств отображаются: нормаль, площадь, количество треугольников

---

## Панель инструментов

### Примитивы
| Кнопка | Действие |
|--------|----------|
| Куб | Создать куб 1×1×1 |
| Сфера | Создать сферу радиусом 0.5 |
| Цилиндр | Создать цилиндр радиусом 0.5, высотой 1 |
| Конус | Создать конус радиусом 0.5, высотой 1 |

### Булевы операции
Требуют выделения 2 объектов.

| Кнопка | Действие |
|--------|----------|
| Union (∪) | Объединение |
| Difference (−) | Вычитание (первый минус второй) |
| Intersection (∩) | Пересечение |

### Эскизы
| Кнопка | Действие |
|--------|----------|
| Эскиз XY | Создать эскиз на плоскости XY |
| Эскиз XZ | Создать эскиз на плоскости XZ |
| Эскиз YZ | Создать эскиз на плоскости YZ |

### Операции с эскизами
Требуют выделения эскиза.

| Кнопка | Действие |
|--------|----------|
| Extrude | Выдавить профиль |
| Revolve | Вращать профиль вокруг оси |

### Cut (Вырезание)
Требует выделения эскиза и целевого объекта.

| Кнопка | Действие |
|--------|----------|
| Cut | Вырезать форму эскиза из объекта |

## Редактирование эскиза

### Вход в режим редактирования
- **Двойной клик** на эскиз
- Или **правый клик → Редактировать**

### Инструменты эскиза

| Инструмент | Описание |
|------------|----------|
| Линия | Два клика: начало и конец |
| Прямоугольник | Два клика: угол и противоположный угол |
| Круг | Два клика: центр и точка на окружности |
| Дуга | Три клика: центр, начало, конец |
| Полилиния | Несколько кликов, правый клик — завершить |
| Сплайн | Несколько кликов, правый клик — завершить |

### Выход из режима
- **Esc** или кнопка "Готово"
- Или **правый клик → Завершить редактирование**

---

## Эскиз на грани

1. Выделите грань (Shift+клик)
2. Правый клик → "Эскиз на грани"
3. Камера автоматически повернётся перпендикулярно грани
4. Нарисуйте нужные элементы
5. Выйдите из режима эскиза

### Создание отверстий
1. Создайте эскиз на грани объекта
2. Нарисуйте круги (или другие формы)
3. Выделите эскиз и целевой объект (Ctrl+клик)
4. Нажмите "Cut"
5. Установите глубину в панели свойств

---

## Контекстное меню (правый клик)

### На объекте
- **Фокус** — центрировать камеру на объекте
- **Скрыть** — скрыть объект
- **Дублировать** — создать копию

### На эскизе
- **Редактировать** — войти в режим редактирования
- **Завершить редактирование** — выйти из режима

### На выделенной грани
- **Эскиз на грани** — создать эскиз на этой грани

---

## Панель свойств

Отображает параметры выделенного объекта.

### Примитив
- Position (X, Y, Z)
- Rotation (X, Y, Z)
- Scale (X, Y, Z)
- Параметры примитива (радиус, высота и т.д.)

### Эскиз
- Плоскость (XY, XZ, YZ)
- Смещение
- Количество элементов

### Extrude
- Высота

### Cut
- Глубина
- Целевой объект

### Грань
- Нормаль
- Площадь
- Количество треугольников

---

## Дерево сцены

Отображает иерархию объектов.

- **Клик** — выделить объект
- **Глаз** — показать/скрыть
- Consumed объекты показаны серым (использованы в операциях)

## Советы

1. **Множественный Cut**: Нарисуйте все отверстия в одном эскизе — они вырежутся одновременно

2. **Точное позиционирование**: Используйте панель свойств для ввода точных координат

3. **Рисование на грани**: Убедитесь, что камера смотрит перпендикулярно — это происходит автоматически при создании эскиза на грани

4. **Булевы операции**: Порядок выделения важен для Difference — первый объект минус второй
