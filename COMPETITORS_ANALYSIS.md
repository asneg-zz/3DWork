## Российский рынок — Обзор

### Рынок 3D-печати в России
- **Объём 2024:** ~6,5-7 млрд ₽ (рост 35% г/г)
- **Прогноз 2030:** ~14,5 млрд ₽ (CAGR 20%)
- **Доля в мире:** <1%, но растёт в 10 раз за 8 лет
- **Драйверы:** импортозамещение, госзаказы, промышленность

### Рынок САПР в России
- **Объём 2024:** ~34 млрд ₽
- **Рост:** +21% г/г
- **Лидеры:** АСКОН (6,6 млрд), Нанософт (4,5 млрд), Топ Системы

### Особенности российского рынка
- Уход западных вендоров (Autodesk, Dassault, PTC)
- Активное импортозамещение
- Госпрограммы поддержки отечественного ПО
- Рост спроса на облачные решения
- Санкции → невозможность использовать Stripe, AWS

## Детальный анализ конкурентов

### 1. Onshape (PTC)

**Цены:**
- Free: публичные проекты, некоммерческое использование
- Standard: $1,500/год ($125/мес)
- Professional: $2,100/год ($175/мес)
- Enterprise: индивидуально

**Сильные стороны:**
- Профессиональный параметрический CAD
- Полноценные эскизы с ограничениями
- История версий и ветвление (как Git)
- Коллаборация в реальном времени
- STEP/IGES экспорт
- PDM встроен

**Слабые стороны:**
- Очень дорого для хоббистов
- Избыточен для 3D-печати
- Нет инструментов подготовки к печати
- Сложный интерфейс для новичков
- Бесплатный план — только публичные проекты

**Вывод:** Не конкурент для рынка 3D-печати из-за цены и сложности.

---

### 2. Tinkercad (Autodesk)

**Цены:**
- Полностью бесплатно

**Сильные стороны:**
- Абсолютно бесплатно
- Очень простой интерфейс
- Отлично для детей и образования
- Быстрый старт
- Большое сообщество

**Слабые стороны:**
- Нет параметрического моделирования
- Нет эскизов и ограничений
- Только примитивы + булевы операции
- Нет истории операций (дерева фич)
- Невозможно редактировать после создания
- Нет STEP экспорта
- Ограничен для сложных моделей

**Вывод:** Хорош для простых моделей, но потолок очень низкий.

---

### 3. SelfCAD

**Цены:**
- Free: ограниченные функции
- Pro: $14.99/мес или $139.99/год
- Perpetual: $599 единоразово

**Сильные стороны:**
- Встроенный слайсер
- Скульптинг
- Анимация
- Доступная цена
- Работает в браузере и оффлайн
- Интеграция с MyMiniFactory

**Слабые стороны:**
- Слабое параметрическое моделирование
- Нет полноценных эскизов с ограничениями
- Интерфейс перегружен
- Производительность на сложных моделях

**Вывод:** Прямой конкурент, но слабее в CAD-функциях.

---

### 4. Vectary

**Цены:**
- Starter: бесплатно (с водяным знаком)
- Pro: $12-15/мес
- Enterprise: индивидуально

**Сильные стороны:**
- Красивый современный интерфейс
- AR интеграция
- Хорош для презентаций
- Простой в освоении
- AI генерация

**Слабые стороны:**
- Не CAD — больше для дизайнеров
- Нет параметрического моделирования
- Нет инструментов для 3D-печати
- STEP только в Enterprise

**Вывод:** Не конкурент — другая ниша (визуализация, маркетинг).

**Слабые стороны:**

- Очень дорого для хоббистов
- Нет облачной версии
- Избыточен для 3D-печати
- Сложный интерфейс

**Вывод:** Не конкурент для SaaS — только desktop, дорого.

## SWOT-анализ нашего проекта

### Сильные стороны (Strengths)

| Преимущество | Описание |
|--------------|----------|
| **Параметрическое моделирование** | Полноценные эскизы с ограничениями — как в SolidWorks |
| **Фокус на 3D-печать** | Инструменты специально для подготовки к печати |
| **Rust backend** | Высокая производительность, надёжность, безопасность |
| **Современный стек** | Легко масштабировать и поддерживать |
| **Доступная цена** | $9-29 против $125+ у Onshape |
| **История операций** | Полноценное дерево фич с undo/redo |
| **STEP экспорт** | В отличие от Tinkercad и SelfCAD |

### Слабые стороны (Weaknesses)

| Недостаток | План решения |
|------------|--------------|
| **Новый продукт** | Агрессивный маркетинг, бесплатный план |
| **Нет сообщества** | Форум, Discord, YouTube туториалы |
| **Меньше функций** | Итеративное развитие по фидбеку |
| **Нет мобильного приложения** | Web-first, responsive дизайн |

### Возможности (Opportunities)

| Возможность | Как использовать |
|-------------|------------------|
| **Рост рынка 3D-печати** | Специализация именно на этой нише |
| **Недовольство Tinkercad** | Привлечь "выросших" пользователей |
| **Цена Onshape** | Предложить 90% функций за 10% цены |
| **Интеграции** | OctoPrint, Klipper, слайсеры |
| **AI тренд** | Генерация моделей по описанию |

### Угрозы (Threats)

| Угроза | Митигация |
|--------|-----------|
| **Autodesk купит нишу** | Скорость развития, уникальные фичи |
| **Onshape снизит цены** | Фокус на 3D-печать, а не общий CAD |
| **Open-source альтернативы** | SaaS удобство, поддержка, интеграции |
